# Aadish Kumar — Portfolio Website

A simple, static portfolio website built with plain HTML, CSS, and JavaScript — no frameworks, no build step. Hosted for free on GitHub Pages.

**Live site:** https://aadish-code-create.github.io/aadishportfolio/

## About

This is my personal data analyst portfolio, showcasing my skills, certifications, and selected projects in Python, SQL, and Business Intelligence.

## Built With

- HTML5 & CSS3
- Vanilla JavaScript (scroll animations via `IntersectionObserver`)
- [EmailJS](https://www.emailjs.com/) — for the contact form and resume request form
- GitHub Pages — free static hosting

## Project Structure

```
aadishportfolio/
├── index.html          # Main page structure
├── style.css           # All styling
├── script.js           # Scroll animations, smooth-scroll buttons, EmailJS logic
├── resume.pdf          # Downloadable resume (sent via email on request)
└── images/
    ├── headshot.jpg
    ├── award-photo.jpg
    ├── signature.png
    ├── certs/           # Certification images
    └── projects/         # Project cover images
```

## Features

- Fully responsive layout (desktop, tablet, mobile)
- Smooth-scroll navigation via "See my work" / "Contact me" buttons
- Scroll-triggered fade-in animations
- Project cards link out to their respective GitHub repositories
- Contact form and resume-request form, both powered by EmailJS
- Clean, dependency-free codebase — easy to edit or extend
