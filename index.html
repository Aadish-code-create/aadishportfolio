About Me

When the raw data is a mystery, stakeholders turn to me. As a Data Analyst, I specialize in transforming complex data into clear, actionable stories: data-driven plans, charts, and advice that help leaders make smart decisions. I chose to start my career in data analytics after watching Michael Burry (played by Christian Bale). I promised myself I'd end up in finance, specifically in the predicting business.

Expertise

Skills

  • SQL / Data Querying Logic — joins, subqueries, window functions, aggregations. The single most-tested skill in DA interviews and screening rounds.
  • Excel Proficiency-Adjacent Logic — data cleaning and wrangling: handling nulls, duplicates, inconsistent formats.
  • Statistics Fundamentals — descriptive stats, correlation vs. causation, basic hypothesis testing.

  • Power BI / Tableau — for building dashboards and visualizations.
  • Git / GitHub — for versioning and showcasing portfolio projects.
